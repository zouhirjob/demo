Une solution de gestion d’API basée sur Azure API Management et le portail développeur implique les acteurs suivants :

Les éditeurs d’API : ce sont les personnes qui publient et gèrent les API dans Azure API Management. Ils utilisent le portail d’administration1 pour configurer les API, les plans, les produits, les politiques, les abonnements, les certificats, les groupes, les utilisateurs et les notifications2.
Les développeurs d’API : ce sont les personnes qui créent et maintiennent les services back-end exposés par les API. Ils peuvent utiliser différents outils et langages pour développer les API, et les déployer dans Azure ou ailleurs3.
Les consommateurs d’API : ce sont les personnes qui utilisent les API publiées dans Azure API Management. Ils peuvent être des développeurs internes ou externes, des partenaires ou des clients. Ils utilisent le portail développeur pour découvrir, tester et consommer les API, ainsi que pour gérer leurs abonnements, leurs clés et leurs profils.
