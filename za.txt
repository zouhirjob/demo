Suppression d’une API:
Objectif: Permettre aux utilisateurs d’effacer une API de l’instance Azure API Management.
Acteurs: Éditeurs d’API, administrateurs.
Scénario:
L’utilisateur accède à l’instance Azure API Management.
Il sélectionne l’API à supprimer.
Il confirme la suppression.
Préconditions: L’API doit être configurée dans l’instance.
Postconditions:
L’API est supprimée de l’instance.
Les métadonnées et les configurations associées à l’API sont effacées.
Exigences:
Confirmation de l’utilisateur avant la suppression.
Exceptions:
Si l’API est supprimée accidentellement, la fonctionnalité de suppression réversible peut être utilisée pour récupérer l’API.
